Grey Store - Web Design & Development

Tentang Project

Grey Store adalah sebuah toko online yang menjual berbagai Game PC dan Produk digital lainnya. Website ini adalah Website untuk bisnis saya sendiri. Website ini diperuntukan untuk Menjual Jasa Instalasi Game PC, Jasa Download Game PC, Jasa Install Windows dan lain sebagainya, nantinya akan dipromosikan melalui Facebook.


Target Pengguna

Semua orang yang punya PC/Laptop terutama Peminat Game.


Wireframing

Berikut adalah Wireframe lama dari Website Grey Store, karena website ini masih terus dikembangkan hingga sekarang.


UI Design

Berikut adalah hasil desain UI untuk Grey Store yang telah jadi.


Development

Website Grey Store ini dikembangkan menggunakan HTML, CSS, Javascript, SASS, SwiperJS, dan Bootstrap 4. Untuk mengunjungi Website nya anda bisa klik Tombol dibawah ini.
